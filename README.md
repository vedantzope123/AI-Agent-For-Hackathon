   MediCare AI | Agentic Nurse Assistant
 
 MediCare AI is a lightweight, responsive web application powered by Google Gemini AI. It serves as an "Agentic Nurse Assistant" designed to help users assess symptoms, provide home remedy suggestions, and identify potentially critical medical conditions.

The application includes built-in clinical tools for generating medical summaries and recovery diet plans, all wrapped in a professional, modern interface.

‚ú® Key Features

AI Symptom Assessment: Real-time chat interface with "CuraFlow," an empathetic AI nurse agent.

Intelligent Triage: Automatically detects serious symptoms and flags them as ||CRITICAL_CONDITION||, triggering a priority appointment workflow.

Clinical Summary Tool: Generates professional medical scribe notes (Chief Complaint, HPI, Assessment, Plan) from the chat history.

Recovery Diet Planner: Provides a 1-day meal plan tailored to the symptoms 
discussed in the chat.

Modern UI/UX: Built with Tailwind CSS, featuring smooth animations, mobile-responsive design, and a dedicated settings portal.

Privacy Focused: API keys are stored locally in the session and are not saved to any external database.
